# Resources

Useful references for IBM PC serial communication and file transfer.

## Kermit

- [Columbia University Kermit Project](https://www.columbia.edu/kermit/) - The official Kermit archive
- [MS-Kermit Downloads](https://www.columbia.edu/kermit/mskermit.html) - MS-Kermit for DOS
- [KERMIT.BAS](https://www.columbia.edu/kermit/ftp/old/basic/kermit.bas) - Receive-only Kermit in BASIC

- [RS-232 Wikipedia](https://en.wikipedia.org/wiki/RS-232) - Overview of the standard
